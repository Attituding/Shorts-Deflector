Shorts Deflector
An extension that changes the Shorts UI to the desktop UI
Requests
chrome.declarativeNetRequest
Enables all redirecting except for the Shorts item from the YouTube navigation menu or left-clicking on a Short. This method of redirecting is faster than Page Updates and is used whenever possible.
Page Updates
chrome.tabs
Enables all redirecting except for new tabs, new windows, or external hyperlinks. Enabling the Request option will improve performance for overlapping redirects as this method is slower.
