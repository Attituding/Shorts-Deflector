Shorts Deflector
An extension that changes the Shorts UI to the desktop UI
Requests
browser.webRequest
Enables all redirecting except for the Shorts item from the YouTube navigation menu or left-clicking on a Short. This method of redirecting is faster than Page Updates and is used whenever possible.
Page Updates
browser.tabs
Enables all redirecting in all situations, which includes clicking the Shorts item from the navigation menu or left-clicking on a Short. You should also enable the Requests option for improved performance.
